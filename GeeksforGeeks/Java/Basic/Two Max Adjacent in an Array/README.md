### 📜 Problem Description
<p><span style="font-size: 14pt;">Given an array <strong>arr[]</strong>, find the <strong>maximum </strong>sum of any two adjacent elements in the array.</span></p>
<p><span style="font-size: 14pt;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 14pt;"><strong>Input:</strong> arr[] = [1, 2, 3, 4, 5]
<strong>Output:</strong> 9
<strong>Explanation: </strong>The maximum adjacent sum is 4 + 5 = 9.</span></pre>
<pre><span style="font-size: 14pt;"><strong>Input:</strong> arr[] = [2, 1, 3, 4]
<strong>Output:</strong> 7
<strong>Explanation: </strong>The maximum adjacent sum is 3 + 4 = 7<span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">.</span></span></pre>
<p><span style="font-size: 14pt;"><strong>Constraints:</strong><br>2 ≤ arr.size() ≤ 10<sup>6</sup><br>0 ≤ arr[i] ≤ 10<sup>5</sup></span></p>
